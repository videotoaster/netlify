Prerequisites

  • A Linux machine
    Can be a Raspberry Pi, an old Optiplex, or an old laptop. Just a semi-competent computer.
    psst, you can do this on your machine, too!
  • A Debian installation
    Also works just fine with Raspbian! But burn a disc or make a Debian thumbdrive to install.
  • Some knowledge of how Linux works
    I'm too incompetent to describe the entirety of how Linux works.
  • Wi-fi, or Ethernet
    If you don't have a wireless connection, you can go ethernet. Just make sure it is connected!

Step 1

Install nginx

Run the following command on your server machine:
sudo apt install nginx fcgiwrap
The installation should work properly. If not, try sudo apt update and try again.
